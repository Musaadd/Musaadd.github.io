Ultra-realistic Voice Generation and Text Annotation System for Quranic Translations

This project focuses on creating an ultra-realistic voice generation (المرتل الذكي) and text annotation system to enhance Quranic translations. It aims to help people by making their lives easier and improving accessibility to the Quranic texts in various languages.

The system uses modle for speech synthesis technologies to generate natural-sounding voice for Quranic recitations. It also incorporates text annotation for taging words of translation Quran.
